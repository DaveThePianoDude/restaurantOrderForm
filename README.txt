Example of Cross-Site Web Service using WCF, JSONp, Ajax, CSS.

To Execute:

1.  Launch IDE.
2.  Set MenuWcfRestService as Startup project
3.  Run w/o debugging.
4.  Return to IDE.
5.  Set RestaurantSite as Startup project and order_dev.htm as startup file.
6.  Create C:\RestaurantOrders folder.
7.  Run w/o debugging.
8.  Place your orders!
